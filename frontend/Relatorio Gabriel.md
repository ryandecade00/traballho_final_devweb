Desenvolvimento Frontend.

Fiquei responsável pela maior parte do Frontend, especialmente a estilização utilizando HTML, React e tivemos que migrar para CSS puro pois enfrentamos muitos problemas com o Tailwind e o tempo estava encurtando. Desenvolvi a interface visual da aplicação, cuidei dos componentes principais, organização das telas, responsividade e navegação entre as páginas.

O foco foi garantir uma interface simples, intuitiva e funcional para o usuário, mantendo uma boa experiência visual. Mesmo assim, em várias partes relacionadas a JavaScript nós colaboramos juntos, trocando ideias principalmente sobre rotas, estados e consumo da API.

Apesar de ao chegar no final do projeto com alguns problemas de vulnerabilidade acho que a proposta da atividade foi cumprida, gostamos do resultado final e ainda pensamos em polir o projeto até ao 100%.
